# Task - Extensión de Visual Studio Code


## Descripción
Task es una extensión para Visual Studio Code que te permite gestionar y organizar pequeñas tareas directamente dentro del editor. Con esta extensión, puedes crear, editar, marcar como completadas y eliminar tareas de forma rápida y sencilla, lo que te ayuda a mantener un seguimiento de tus pendientes mientras trabajas en tu código.

## Características
- Crear nuevas tareas.
- Editar el contenido de las tareas.
- Marcar las tareas como completadas.
- Eliminar tareas.
- Interfaz fácil de usar e integrada en Visual Studio Code.

## Instalación
1. Abre Visual Studio Code.
2. Ve a la pestaña de Extensiones (Ctrl+Shift+X).
3. Busca "Task" en el Marketplace.
4. Haz clic en "Instalar".
5. Una vez instalada, reinicia Visual Studio Code si es necesario.

## Uso
1. Abre un archivo en Visual Studio Code.
2. Abre la vista de tareas utilizando el comando `Task: Open Tasks` desde el menú de comandos o la paleta de comandos (Ctrl+Shift+P).
3. Aquí puedes crear nuevas tareas, marcarlas como completadas, editarlas o eliminarlas según sea necesario.
4. Las tareas se guardan automáticamente y están disponibles cada vez que abras el archivo.
